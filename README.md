# StreamDeck Timer Service

A web service that enables StreamDeck users to create and track custom timers with a built-in history view.

## Features
- Create timers via URL endpoints
- View timer history with sorting/filtering
- Export timer data to CSV/Excel
- Clear timer history
- User-friendly setup instructions

## Setup
1. Install Docker and Docker Compose
2. Clone repository
3. Run:
```bash
docker-compose build
docker-compose up -d
